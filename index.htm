MCOS is an operating system designed for use with ComputerCraft. This operating system was designed with simplicity in mind so you can get the best experience out of the

system as you can.

 

The operating system comes jampacked with:

- Functioning Shell System with command 'exit' to return to the desktop.

- LuaIDE

- Firewolf

- Settings

Our current settings are only Custom Boot Screen. That setting is very buggy and WILL open up vunerabilies to your system unless you use os.pullEvent = os.pullEventRaw
